# Front-End-em-Java
Projeto de estudos simples sistema desktop em Java (Swing) para cadastro e compra de pacotes de viagem, com tela de login, criação de conta, validação de dados do usuário, formulário completo de cadastro, navegação entre janelas e módulo de suporte. Utiliza herança, eventos, layouts personalizados e controle básico de usuários em memória.
